SAHA Expo Web Scraper

This project extracts exhibitor data from the SAHA Expo website.

Features:
-Scrapes multi-page listings
-Extracts detailed exhibitor information
-Handles missing data fields
-Decodes protected email addresses
-Outputs structured CSV data


Data Extracted:
-Company Name
-Hall & Stand
-Industry
-Email
-Website
-Address
-Country
-About Description


Tech Stack:
-Python
-Requests
-BeautifulSoup
-Pandas


Output:
-Clean dataset ready for analysis or business use.
